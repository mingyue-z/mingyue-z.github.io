# Academic Homepage for GitHub Pages

This is a simple multi-page academic personal website built with plain HTML, CSS, and a small amount of JavaScript.

## Pages

- `index.html`: About Me
- `research.html`: Research
- `teaching.html`: Teaching

## How to Customize

1. Open the HTML files and replace the sample name, university, email, office, research topics, courses, and publication list.
2. If you want to change the visual style, edit `styles.css`.
3. If you want your homepage URL to be the repository root, keep `index.html` as the main entry page.

## Deploy to GitHub Pages

1. Create a new repository on GitHub.
2. Push this project to that repository.
3. On GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your main branch and the `/ (root)` folder, then save.
6. Wait a minute or two for GitHub Pages to publish the site.

Your site URL will usually look like:

- `https://your-username.github.io/repository-name/`

If you want to use `https://your-username.github.io/`, create a repository named:

- `your-username.github.io`
